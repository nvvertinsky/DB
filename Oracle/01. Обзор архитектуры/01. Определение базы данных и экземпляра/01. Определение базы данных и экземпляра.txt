База данных - коллекция физических файлов или дисков операционной систем.

База данных с единсветнным владельцем (single tenant) - это обособленный набор файлов данных, управляющих файлов, журнальных файлов для восстановления, файлов параметров, который в добавок ко всем метаданным, данным и коду приложения содержит все метаданные Oracle (например all_objects), данные Oracle и код Oracle. До 12с - это был единственный тип базы данных.

Контейнерная база данных - это обособленный набор файлов, который включает только метаданные Oracle, данные Oracle и код Oracle. 

Подключаемая база данных - это набор лишь файлов данных. Файлы данных содержат только метаданные приложения и код приложения.

Экземпляр - набор фоновых процессов или потоков и области разделяемой памяти Oracle.

Одна контейнерная БД может работать только с одним экземпляром. И к ним можно подключать несколько подключаемых баз данных.
